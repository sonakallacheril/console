Console Web App

Project On Console

Go to code.visualstudio.com and download it. Then, go to extesions. Next, find live preview in extensions. After that, make a folder on your desktop and go to explorer. Then, select the folder you want to use, and right click on the folder name to create a new file Make three new files called: index.html, style.css, and main.js. Then, start coding.

To print anything on console you write something like this in main.js:

var example = 5;

console.log(example);
